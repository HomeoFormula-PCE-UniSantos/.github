## 📋 Sobre a Organização
A **Plataforma HomeoFormula** é uma iniciativa desenvolvida para a Farmácia de Manipulação Homeofórmula (Santos - SP). Nosso objetivo é otimizar e digitalizar o fluxo de atendimento da empresa, centralizando as interações em um ambiente escalável que proporciona maior autonomia para os clientes e eficiência operacional para a equipe.
Atuamos em uma aplicação que permite que os usuários realizem suas demandas essenciais de forma 100% autônoma (Autoatendimento, Renovação 1 Clique e Gestão de Grupo Família).
Pretendemos aumentar o escopo para integrar 100% com o erp da farmácia.
